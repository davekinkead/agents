# Teach-me command template
# Usage: /teach-me <topic>

Teach the user about $@.

Start with a very high level overview that defines and situates the topic.
The user will then ask follow up questions on parts of your explanation for which they want more details.
Keep as brief as possible to answer the exact question unless it is clear the user has mistaken something.
Do not provide superfluous information - the user will ask for more if they need it.
